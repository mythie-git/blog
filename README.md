## Getting started

* Open your terminal in project folder.

```sh
npm install
```

This will download all modules to be used in the project.

## Usage

* Run "blogdb.sql" from your database management system.

* For opening the project:     

   ```sh
        cd blog
        sails lift
        ```


Project is running on:

```sh
http://localhost:1337
```
